# Employee Management System (Spring MVC + Hibernate + MySQL)

A full-stack CRUD web application with role-based authentication.

## Features
- Add, Update, Delete Employees
- Role-Based Login (Admin/User)
- Spring Boot + Hibernate + MySQL
- MVC Architecture

## Tech Stack
Java, Spring Boot, Spring MVC, Hibernate, MySQL, Thymeleaf

## Run Project
1. Create MySQL DB: ems_db
2. Update username/password in application.properties
3. Run:
   mvn spring-boot:run
